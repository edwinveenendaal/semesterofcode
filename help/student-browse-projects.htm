Choosing a project and writing a proposal

 

You can browse the list of current projects from either the front page link 'projects' or from within your dashboard.

The list is sortable by column and you can limit the project ideas by particular organisations. Additionally you can also search by keyword or tags, (i.e. java or c++ etc.)

Clicking on the project title should open that project description in a dialog window...

As well as the project description, you can also see some statistical information.

  • How many other students have submitted a proposal for this project idea.
  • If the project creator has offered this project to another student, or if he/she has marked another proposal as their interim favourite.
  • An expandable comments section where you can ask questions. The owner of this project idea will be emailed any comment you add here.
  • A button 'Mark this project' which allows you to bookmark project ideas you are interested in, allowing you to return and revisit them later. (Note: These bookmarked projects are subsequently available by clicking on the 'dashboard' link and then choosing 'List favourite projects'.)

You can click 'Prev' and 'Next' at the bottom of the screen to navigate through the current resultset. (Note: you will have to close this window and go to the next page in order to progress to the next set of project ideas as they are returned in a page format.)
