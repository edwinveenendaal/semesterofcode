How to register additional organisation administrators and mentors for your organisation

The organisation administrator role in the VPS is intended to help in managing the tasks related to the organisation. They are also mentors, but will slightly elevated privilleges. Only the organisation administrator can invite new mentors and admins into their organisation. Mentors cannot.

To invite additional organisation admins and new mentors to your organisation, navigate to your Dashboard and click on the Managed organisations link.

Scroll down and you will also see the Code field. Of course a fake code is shown here (normally 9 characters). The code you will see, is the code to send to new users you wish to add/register as new mentors. Additionally, you can click the Invite mentor button which will bring up a form which allows you to enter email addresses to which a preformatted email will be sent containing the code.

Further down you will also see the Owner code field. The actual raw code is shown here (9 characters) This is the code to send to new users you wish to add/register as additional organisation administrators. Additionally, you can click the Invite organisation administrator button which will bring up a form which allows you to enter email addresses which a preformatted email will be sent to containing the code.
