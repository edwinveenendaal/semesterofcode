Browsing proposals made by students

 

As students start to write proposals, you will be sent email messages to let you know each time a new one has been added or modified. To view the proposal you should navigate to the dashboard and choose the Proposals submitted by students from my institute and in my groups link.

For each of your students there should be a entry shown, which shows how many proposals the student has created. If there are more than zero proposals for a given student, there will be an info icon you can click to list the proposals made by that particular student.

Click on the info icon under the 'view' column for a particular student. Now you should see a list of proposals made by that student. Note: if a student's proposal is marked as 'Draft' you can view it, but the project ideas creator/mentor cannot yet.

To open and view a student's proposal click the info icon for that entry under the 'proposal detail' column.

A dialog window should appear containing six tabs.

  • Overview - contains an overview of the proposal and allows you to add comments to it, which in turn are emailed back to the student and mentor.
  • Student Details - lists the student's details, which institution he/she is attending and their supervisors details.
  • Solution Summary - the student's brief proposal synopsis
  • Solution Description - the student's detailed proposal text
  • Project - is a reference back to the project idea initially created by the mentor
  • Status - allows the mentor/project idea creator to set this proposals status and optionally make this student an 'offer'. Here you can view what the current status is.
